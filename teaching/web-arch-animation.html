── Request Lifecycle ──
01 · User Action
User clicks a button or submits a form. JS captures the event.
02 · HTTP Request
JS calls fetch() and sends an HTTP request to the Python API.
03 · Business Logic
Python (Flask/Django) receives the request, validates input, runs logic.
04 · DB Query
Python runs SQL on the SQLite .db file to read or write data.
05 · JSON Response
Python packages results as JSON and returns it over HTTP.
06 · UI Update
JS parses the JSON and updates HTML/CSS — user sees fresh content.
